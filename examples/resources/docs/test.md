# Test Resource

This is a static MCP resource used for testing the lmchatkit `@resource` attachment feature.

## Features

- Markdown is rendered when the model receives it
- The content is injected as context, not displayed as a chat bubble
- The user sees a green attachment chip on their message instead

## Usage

Type `@test` in the chat composer, select `docs://test.md`, then ask the model a question about this content.
